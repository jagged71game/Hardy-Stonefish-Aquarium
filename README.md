# Hardy Stonefish Aquarium

Open `index.html` in a browser. No build step is required.

The game attempts a live, read-only lookup through the public Splinterlands API. If that service is unavailable or blocks browser requests, the demo tank remains fully playable.

Only the first 60 stonefish are drawn, even when a collection contains more; the total collection count is still shown.

The aquarium now includes persistent Tamagotchi-style care. Hunger, happiness, hygiene, health, waste, illness, medicine, and glass damage are saved locally for each keeper name. Needs continue to decay for a limited period while the game is closed.

Use **Full-screen aquarium** for an edge-to-edge ambient watch mode. Its compact Care dock stays closed until needed, then provides live status checks and all aquarium actions. On mobile, the same thumb-friendly dock replaces the large desktop controls.
